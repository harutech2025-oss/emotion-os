Stato
오늘의 상태를 읽고, 에너지가 새는 지점을 찾아, 하루를 다시 운영하게 만드는 앱
Powered by Emotion OS · by HaruTech Lab · Emotional Engineering Institute
---
감정 누수를 10%만 줄여도, 하루의 질은 달라집니다.
Stato는 감정공학 기반의 하루 운영 앱입니다.
2분 점검 → 상태 해석 → 실행 패치 → 백과사전 연결까지 하나의 생태계로 이어집니다.
🔍 상태 점검 (Full Scan)
21문항 자기 점검으로 현재 감정 운영 상태를 분석합니다.
📊 7가지 감정 누수 유형 (Q1~Q7)
에너지가 새는 대표 패턴을 식별합니다.
Q1 조급형 · Q2 과민형 · Q3 회피형 · Q4 분출형
Q5 우울형 · Q6 비교형 · Q7 통제형
🐛 버그 감지
지금 시스템에서 활성화된 대표 버그를 사용자 언어로 보여줍니다.
무한 불안 루프 · 입력 과부하 · 감정 억제 누적 · 과열 반응 폭주
시동 실패 · 저출력 고착 · 침대 고착 · 자기비하 증폭
비교 누수 · 완벽주의 통제 과잉
🩹 실행 패치 (Hot Fix)
상태에 따른 맞춤형 회복 행동을 타이머 기반으로 제공합니다.
호흡 리셋 · 감각 리셋 · 걱정 덤프 · 감정 라벨링 등 8종
📚 감정공학 백과사전 연결
점검 결과에 대한 이론적 배경과 심화 이해 자료를 제공합니다.
Emotion OS · 5R 기본 구조 · 호모 레귤란스 · Bug/Patch 카드 등 40+ 문서

앱 사용
웹앱: stato.kr
PWA(Progressive Web App)로 제작되어 별도 설치 없이 브라우저에서 바로 사용할 수 있습니다.
모바일 홈 화면에 추가하면 네이티브 앱처럼 작동합니다.

차별점
기존 감정/명상/치유 앱이 인식 · 공감 · 안정 · 머무름에 무게를 둔다면,
Stato는 운영 · 실행 · 감정 누수 차단 · 주의력 회복 · 일상 성능 회복에 무게를 둡니다.
가장 큰 차별점은 점검 → 해석 → 실행 → 백과사전 연결이 하나의 생태계로 이어지는 구조입니다.
